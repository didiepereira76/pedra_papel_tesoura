# Pedra, Papel e Tesoura - Edição Esports 🎮

Um clássico jogo de Pedra, Papel e Tesoura recriado com uma estética moderna, agressiva e imersiva inspirada no universo dos esportes eletrônicos (Esports).

## 🚀 Sobre o Projeto
Este projeto pega a lógica simples de um clássico e a eleva utilizando técnicas modernas de CSS e manipulação de DOM com JavaScript. O resultado é uma interface com cara de "telão de campeonato", com direito a placar gigante, arena de embate entre as jogadas e luzes Neon que destacam o vencedor da rodada.

### Principais Funcionalidades
- **Arena Versus (VS):** As jogadas se enfrentam de forma visual no centro da tela.
- **Feedback Visual Neon:** O lado do vencedor na Arena cresce e emite um brilho intenso (Azul para o Jogador, Vermelho para o Computador).
- **Botões Estilo Fliperama:** Controles responsivos em 3D que "afundam" quando são clicados, simulando o feedback tátil de um controle de fliperama.
- **Placar Interativo:** Um "Scoreboard" superior usando a tipografia impactante Teko.
- **Reset Rápido:** Um botão discreto para zerar o placar e limpar a arena a qualquer momento para uma nova partida.

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica, contêiner de Placar e Arena de Batalha.
- **CSS3:** Animações (`transform`, `scale`, `translate`), variáveis e efeitos Glow Neon (`box-shadow` e `text-shadow`), e Web Fonts (Google Fonts).
- **JavaScript Vanilla:** Lógica do jogo (Math.random), manipulação do DOM e injeção de classes CSS dinâmicas (`classList.add()`).

## ⚙️ Como Executar
O projeto roda diretamente no navegador, sem a necessidade de nenhum servidor, build ou instalação.

1. Clone ou faça o download deste repositório.
2. Extraia os arquivos em uma pasta na sua máquina.
3. Dê um duplo clique no arquivo `index.html` para abri-lo no seu navegador de preferência.
4. Clique em Pedra, Papel ou Tesoura para iniciar a batalha contra a máquina!

## 📸 Estrutura de Arquivos
- `index.html` - Toda a estrutura visual (Placar, Arena, Controles).
- `style.css` - O design Esports e animações.
- `index.js` - O cérebro por trás da escolha do computador e atualização das luzes.
